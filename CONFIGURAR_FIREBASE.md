# Firebase (histórico / opcional)

O **backend de dados e auth do produto** é **Supabase**, não Firebase. Veja **`CONFIGURAR_SUPABASE.md`** e **`docs/overview.md`**.

O SDK Firebase no repositório é **opcional** e legado para alguns caminhos; para desenvolvimento alinhado ao produto atual, não dependa da documentação antiga só sobre Firebase.

A cópia arquivada do guia anterior está em **`docs/archive/root-legacy/CONFIGURAR_FIREBASE.md`** (consulta apenas para arquivo).
