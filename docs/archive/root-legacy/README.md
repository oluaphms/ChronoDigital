Este diretório concentra ficheiros **`.md`** que estavam na **raiz** do projeto e foram movidos aqui (**abril de 2026**) para:

- Evitar inconsistência entre dezenas de notas pontuais (Firebase, AdminView, otimizações, deploys pontuais) e a documentação **canônica** em **`docs/`** + **`README.md`**.
